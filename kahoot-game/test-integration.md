# 🧪 Ricky 遊戲小舖完整整合測試

## 📋 測試準備

### 1. 啟動後端服務器
```bash
cd kahoot-game/backend
go run cmd/main.go
```
服務器應該在 `http://localhost:8080` 啟動

### 2. 啟動前端開發服務器
```bash
cd kahoot-game/frontend
npm install
npm run dev
```
前端應該在 `http://localhost:5173` 啟動
